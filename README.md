# Needle

Needle is a small macOS menu bar app that controls Spotify and shows a draggable frosted-glass now-playing overlay.

## Run

```sh
./script/build_and_run.sh
```

Useful modes:

```sh
./script/build_and_run.sh --install
./script/build_and_run.sh --verify
./script/build_and_run.sh --logs
```

The install mode copies the built app to `~/Applications/Needle.app`.
